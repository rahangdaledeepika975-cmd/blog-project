Blog Project using PHP and MySQL

This is a simple blog application developed using PHP and MySQL as part of an internship task.

# Features
- Create and manage blog posts
- Store posts in MySQL database
- Display posts dynamically on the browser
- Database connection using PHP (mysqli)
- Local server setup using XAMPP

# Technologies Used
- PHP
- MySQL
- XAMPP (Apache & phpMyAdmin)
- HTML (basic)

# Project Structure
- db.php – Database connection file  
- add_post.php – Add new blog posts  
- view_posts.php – Display all blog posts  

# How to Run the Project
1. Install XAMPP and start Apache & MySQL
2. Place the project folder inside `htdocs`
3. Create a database named `blog` in ph
